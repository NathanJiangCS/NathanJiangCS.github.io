    LibEstimote is a location-based information system for public libraries made using Estimote Beacons. The phone app, which interacts with a grid of Estimote Beacons placed throughout the library, provides users contextual information such as books nearby and directions to navigate to the books they want.
    Java, Estimote API

    Project Twice is multifaceted program written completely in Python that is comprised of three main components: a Web Scraper, a Facial Detection system, and Facial Identification interface. It scrapes numerous Twice fansites for images and runs the images through a facial identification program to identify the members.
    Python, BeautifulSoup, OpenCV, FaceRec

    Myusicality is a real-time, gesture-controlled music player/editor that incorporates Myo Armband technology. While playing a song, the user is able to split the audio into multiple frequency bands and, through gestures, control the volume associated with each frequency, effectively mixing the song to the user's preference.
    C++, Python, Pyo, Tkinter, Myo API
